Aplikacja ma służyć do zarządzania bazą danych użytkowników.

Każdy użytkownik powinien zawierać:

Imię, Nazwisko, Nick i Adres Email


Aplikacja powinna udostępnić REST API które pozwala na:

- dodanie nowego użytkownika

- edycję

- usunięcie użytkownika

- wyszukanie po id

- wyszukanie po nicku

- wyszukanie po podanej frazie która może znajdować się w Imieniu, Nazwisku lub Nicku użytkownika
